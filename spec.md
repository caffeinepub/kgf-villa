# Specification

## Summary
**Goal:** Build a single-page static wedding venue website for KGF Villa with an elegant, warm visual theme.

**Planned changes:**
- Sticky navigation bar with "KGF Villa" branding and anchor links (Home, About, Contact); collapses to hamburger menu on mobile
- Hero section with full-width background image, "KGF Villa" heading, tagline, and WhatsApp CTA button
- About / Venue Highlights section with 3–4 feature cards (icons + descriptions) covering halls, gardens, catering, and event management
- Contact section displaying phone number (+91 73559 36696), a green WhatsApp button linking to `https://wa.me/917355936696?text=Hello%2C%20I%27m%20interested%20in%20booking%20KGF%20Villa`, an embedded Google Maps iframe for "Medical Chauraha, inside KP Ground, George Town, Prayagraj, Uttar Pradesh 211002", and the full address as readable text
- Footer with business name, address, phone number, WhatsApp link, and copyright notice
- Consistent warm palette (ivory, gold, warm neutrals), serif headings, sans-serif body, fully responsive layout

**User-visible outcome:** Visitors can learn about KGF Villa, view venue highlights, find the location on a map, and contact the venue directly via WhatsApp — all on a single elegant, mobile-friendly page.
